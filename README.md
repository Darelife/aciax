## JSON Formats
### public/database.json
```json
{
  "branchName Eg: cse":{
    "subjectId Eg: csf111": {
      "Topic Headers: Eg: Handouts": {
        "Link Text": "Link URL"
      }
    }
  }
}
```

### public/checklist.json
```json
{
  "branchName Eg: cse":{
    "subjectId Eg: csf111": {
      "name":"Subject Name Eg: C Programming",
      "Checklist": {
        "Header Eg:Introduction": [
            "Topic 1",
            "Topic 2"
        ]
      }
    }
  }
}
```

### public/subjectNames.json
```json
{
  "cse":{
    "subjectId Eg: csf111": "subjectName Eg: C Programming"
  }
}
```
